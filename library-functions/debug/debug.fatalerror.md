# Debug.FatalError

## `Debug.FatalError()`

Stoppt das Spiel und lässt ein Fenster mit der Fehlermeldung öffnen: "Error: SCRIPT: Debug.FatalError() called."

#### Rückgabewert

none

#### Beispiel

```lua
-- stoppt das spiel und wirft einen FE
Debug.FatalError()
```
