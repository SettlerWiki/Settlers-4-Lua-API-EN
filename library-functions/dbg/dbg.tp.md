# dbg.tp

## `dbg.tp(string)`

Ähnlich wie, jedoch schreibt diese einen String in das für den Spieler nicht sichtbare Konsolenfenster.

Der Kürzel steht wahrscheinlich für trace print (siehe Debug table)

#### Rückgabewert

none

#### Beispiel

```lua
dbg.tp("Hello world")
dbg.tp(0 .. " people will see this")
```
